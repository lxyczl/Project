# English-Specific Rewrite Techniques

## Passive → Active Conversion Rules

### Common Passive Constructions

| Passive Form | Active Alternative |
|-------------|-------------------|
| It was found that... | We found that... / The results show... |
| It has been shown that... | Research demonstrates... / Studies confirm... |
| It is believed that... | Researchers believe... / The community accepts... |
| It is considered to be... | It is... / Experts regard it as... |
| The experiment was conducted | We conducted the experiment |
| The data were analyzed | We analyzed the data |
| The results were obtained | We obtained the results |
| The model was trained | We trained the model |

### When to Keep Passive Voice

Keep passive when:
- The actor is unknown or unimportant: "The samples were collected in 2023"
- Standard methodology description: "The solution was heated to 100°C"
- Emphasizing the action/result: "Significant improvements were observed"
- Method section conventions: "The experiments were performed in triplicate"

### Irregular Past Participle Reference

Common irregular forms that look like past tense but are actually past participles:

| Base Form | Past Participle | Example (passive) |
|-----------|----------------|-------------------|
| show | shown | It was shown that... |
| give | given | The results were given in Table 1 |
| take | taken | Samples were taken at intervals |
| make | made | Adjustments were made to the model |
| know | known | This phenomenon is known to occur |
| find | found | The method was found to be effective |
| think | thought | This approach was thought to be superior |
| say | said | It was said that... |
| do | done | The work was done in stages |
| go | gone | The changes have gone unnoticed |
| come | come | The results have come as expected |
| become | become | This has become a standard approach |

## Nominalization Restoration Rules

### Common Nominalization Patterns

| Nominalized Form | Verb Restoration | Example |
|-----------------|-----------------|---------|
| implementation | implementing | The implementation of the method → Implementing the method |
| analysis | analyzing | The analysis of the data → Analyzing the data |
| development | developing | The development of the model → Developing the model |
| evaluation | evaluating | The evaluation of performance → Evaluating performance |
| investigation | investigating | The investigation of the problem → Investigating the problem |
| consideration | considering | The consideration of factors → Considering factors |
| demonstration | demonstrating | The demonstration of effectiveness → Demonstrating effectiveness |
| observation | observing | The observation of patterns → Observing patterns |
| identification | identifying | The identification of features → Identifying features |
| characterization | characterizing | The characterization of the material → Characterizing the material |
| optimization | optimizing | The optimization of parameters → Optimizing parameters |
| validation | validating | The validation of the model → Validating the model |

### When Nominalization is Acceptable

Keep nominalization when:
- It's a standard technical term: "normalization", "regularization", "classification"
- It improves sentence flow: "The implementation was successful" (vs. "Implementing it was successful")
- It's part of a noun phrase: "data analysis method", "model evaluation procedure"

## Clause Splitting Rules

### Relative Clause → Independent Sentence

**Before:** The model, which was trained on a large dataset, achieved high accuracy.
**After:** The model was trained on a large dataset. It achieved high accuracy.

**Before:** The algorithm that we proposed outperforms existing methods.
**After:** We proposed a new algorithm. It outperforms existing methods.

**Before:** The results, which are presented in Table 1, confirm our hypothesis.
**After:** The results are presented in Table 1. They confirm our hypothesis.

### That/Which/Who Clause Patterns

| Pattern | Split Strategy |
|---------|---------------|
| Noun, which + clause | → Two sentences: "Noun + clause detail. It + main verb..." |
| Noun that + clause | → Two sentences: "We + verb + Noun. It + main verb..." |
| Noun who + clause | → Two sentences: "Noun + clause detail. He/She/They + main verb..." |

## Hedging Removal/Replacement Rules

### Excessive Hedging Patterns

| Hedging Expression | Direct Alternative |
|-------------------|-------------------|
| It seems that... | [State directly] |
| Arguably, ... | [State directly] or "This suggests..." |
| Perhaps, ... | [State directly] or "Possibly, ..." |
| It is possible that... | [State directly] or "This may..." |
| To some extent, ... | [Remove or] "Partially, ..." |
| One might argue... | "We argue..." or [State directly] |
| It could be said that... | "We suggest..." or [State directly] |
| There is a tendency to... | "Researchers often..." or [State directly] |
| It is generally accepted that... | "The field accepts..." or [State directly] |
| It is widely believed that... | "Most researchers believe..." or [State directly] |
| More or less, ... | [Remove] or "Approximately, ..." |
| In a sense, ... | [Remove] or "In this context, ..." |
| Up to a point, ... | [Remove] or "Partially, ..." |
| To a certain degree, ... | [Remove] or "Partially, ..." |

### When Hedging is Appropriate

Keep hedging when:
- Making uncertain claims: "This may explain the results"
- Avoiding overstatement: "These findings suggest rather than prove"
- Respecting limitations: "Within the scope of this study..."
- Academic politeness: "We tentatively propose..."

## Contractions in Academic Writing

### When to Use Contractions

Contractions are acceptable in:
- Informal academic contexts (blog posts, commentaries)
- Direct quotations from informal sources
- Deliberately conversational tone

### When to Avoid Contractions

Avoid contractions in:
- Formal journal articles
- Dissertation/thesis writing
- Conference papers
- Most academic contexts

**Note:** This skill may suggest contractions as an AIGC reduction technique (AI text tends to avoid contractions), but use sparingly and only when context permits.

## Sentence Length Variation Techniques

### Short Sentences (5-10 words)
Use for emphasis, transitions, or breaking monotony.

**Example:** "This is important. It changes everything."

### Medium Sentences (15-25 words)
Standard academic sentence length.

**Example:** "We evaluated the proposed method on three benchmark datasets to assess its generalization capability."

### Long Sentences (30+ words)
Use for complex relationships, but sparingly.

**Example:** "Although the initial results were promising, further investigation revealed that the model's performance degraded significantly when applied to out-of-distribution data, suggesting the need for additional regularization techniques."

### Variation Strategy

Aim for a mix:
- Follow a long sentence with a short one
- Use 2-3 medium sentences, then a short one for emphasis
- Avoid more than 3 consecutive sentences of similar length
